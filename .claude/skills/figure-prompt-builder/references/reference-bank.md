# Bundled Reference Bank

This skill ships with a small bundled reference bank under
`../assets/reference-bank/` so it remains useful in any bootstrapped repo.

Use this bank by default. Reach for the larger repo-level `figure-references/`
archive only when you are working in a repo that actually has it and the
bundled donors are not enough.

## Usage Rule

- pick one structural donor first
- add at most one style donor
- keep result panels as companion donors rather than the main target
