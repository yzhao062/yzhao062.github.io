The 20th ACM Conference on Web Search and Data Mining is open for submissions, with the program organized around the topics shaping today's web systems: https://wsdm-conference.org/2027/

🤖 Foundation Models and Agentic Systems (LLMs for the web, RAG, tool use, agent benchmarks)
🔎 Web Search (ranking, query understanding, dense and sparse retrieval)
💬 Intelligent Assistants (recommendation, conversational systems, personalization)
🕸️ Social Networks (graph mining, GNNs, social network analysis)
🛡️ Privacy, Fairness, Interpretability
✅ Information Integrity (misinformation, content moderation, anomaly and fraud detection)
📊 Web Mining and Content Analysis
📡 Web of Things, Ubiquitous and Mobile Computing
👥 Crowdsourcing and Human Computation
🌟 Emerging and Creative Applications

Three things worth flagging:

✨ Findings track for solid, well-executed work that may not clear the main-conference bar. Papers submitted to the main track are auto-considered, so there is nothing extra to opt into.

🎤 Confirmed keynotes from Bowen Zhou, Lars Backstrom, and Jimmy Huang.
