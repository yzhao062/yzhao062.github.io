# Hashtag sets for X announcements

Pick **3 to 5** per post. Never more. X down-weights stuffed posts.

**Mix across segments:**
- 1 topic-specific (narrow, precise audience)
- 1 to 2 sub-field (medium reach)
- 1 to 2 broad (large but noisier reach)

## By announcement type

### Open-source release
```
Specific topic + #Python + #OpenSource + #MachineLearning (or #DataScience)
```
Working example (PyOD 3): `#AIAgents #AnomalyDetection #MachineLearning #Python #OpenSource`

### Academic paper
```
Topic tag + #MachineLearning + #AcademicTwitter [+ #VenueYear]
```
Working example: `#LLMs #TrustworthyAI #MachineLearning #AcademicTwitter #NeurIPS2026`

### Grant / award
```
#Research + #AcademicTwitter + topic tag + funder tag (optional)
```
Example: `#NSFFunded #TrustworthyAI #AcademicTwitter #MachineLearning`

### Talk / keynote
```
Topic tag + venue tag (#USC, #USCViterbi, conference hashtag)
```
Example: `#LLMs #AIAgents #AcademicTwitter #USCViterbi`

### Student / hiring
```
#AcademicTwitter + #PhDLife + topic tag
```

## By topic

| Bucket | Tags |
|---|---|
| LLM / Agentic | `#AIAgents` `#LLMs` `#LLM` `#RAG` `#PromptEngineering` |
| Anomaly detection | `#AnomalyDetection` `#OutlierDetection` |
| Trust / Safety | `#TrustworthyAI` `#AISafety` `#Robustness` `#AIEvaluation` |
| Core ML / AI | `#MachineLearning` `#DeepLearning` `#AI` |
| Applied / domain | `#FraudDetection` `#Cybersecurity` `#HealthAI` `#Finance` |
| Developer | `#Python` `#OpenSource` `#DataScience` |
| Academic | `#AcademicTwitter` `#PhDLife` `#ResearchPaper` |
| Conferences | `#NeurIPS2026` `#ICML2026` `#ICLR2026` `#KDD2026` `#AAAI2026` |
| Institution | `#USC` `#USCViterbi` |

## Avoid

- `#ChatGPT`, `#OpenAI`, `#Gemini`: attracts bots, drowns signal
- `#Programming`, `#Tech`, `#Coding`: too broad, low conversion
- Spam tags: `#viral`, `#trending`, `#fyp`, `#followback`
- Off-topic mixing: do not pair `#AnomalyDetection` with `#Gaming` just to reach two crowds

## Selection flowchart

```
Start: what is the core topic of this post?
  → Pick 1 specific tag from "By topic" buckets (narrowest bucket that fits).
  → Pick 1 to 2 sub-field tags.
  → Pick 1 to 2 broad tags (#MachineLearning, #AcademicTwitter, #OpenSource).
  → Stop at 5 total.
  → If announcing at a conference: swap one broad tag for the conference tag.
```
